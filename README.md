# Java Spring Boot CRUD Application

## Description

This project is a Java Spring Boot–based CRUD (Create, Read, Update, Delete) application developed to strengthen practical understanding of backend development, RESTful APIs, and database integration.

The application demonstrates fundamental concepts of enterprise application development using Spring Boot and MySQL.

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* IntelliJ IDEA

## Features

* Create, read, update, and delete records
* Role-based structure using predefined roles (ROLE_ADMIN, ROLE_USER)
* RESTful API implementation
* Database integration with MySQL
* Clean and modular project structure

## Setup Requirements

* JDK 8 or higher
* IntelliJ IDEA (or any Java IDE)
* MySQL Server
* Maven

## Purpose

This project was developed for learning and academic practice to enhance backend development skills and understanding of Java Spring Boot–based systems.
